# Release Notes - v0.1.0

Initial release of agcm, a terminal user interface for the Red Hat Customer Portal API.

## Features

- **Case Management**: Browse and view Red Hat support cases with a keyboard-driven interface
- **Case Details**: View case descriptions, comments, and attachments in tabbed panels
- **Filtering**: Filter cases by status, severity, product, and keyword
- **Sorting**: Sort cases by last modified date, created date, severity, or case number
- **Quick Search**: Jump directly to a case by number with `/`
- **Text Search**: Search within case content with `Ctrl+F`
- **Export**: Export individual cases or bulk export all cases to markdown format
- **Mouse Support**: Click to select cases, scroll, switch tabs, and open links
- **Right-click**: Open cases directly in the Red Hat support portal
- **Mask Mode**: Hide sensitive text for screenshots with `--mask` flag

## Keyboard Shortcuts

- `↑/k`, `↓/j` - Navigate up/down
- `←/→` - Switch detail tabs
- `Tab` - Switch between list and detail panes
- `/` - Quick search by case number
- `f` - Filter dialog
- `F` - Clear filter
- `s` - Cycle sort field
- `S` - Toggle sort order
- `e` - Export current case
- `E` - Export all cases
- `r` - Refresh
- `?` - Toggle help
- `q` - Quit

## Installation

Download the appropriate binary for your platform from the releases page, or build from source:

```bash
make build
```

## Configuration

1. Obtain an offline token from https://access.redhat.com/management/api
2. Run `agcm auth login` and paste your token
3. Start the TUI with `agcm`

## Known Limitations

- Requires an active Red Hat subscription with API access
- OAuth token refresh is handled automatically but requires initial offline token setup
